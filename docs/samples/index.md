# TinyBDD Samples

Complete, production-quality examples demonstrating TinyBDD features and industry best practices.

## File-Based DSL Samples

### [Calculator Testing](file-based/calculator.md)

**Complexity**: Beginner

Demonstrates fundamental file-based DSL concepts:
- Creating Gherkin .feature files
- Implementing driver methods with `[DriverMethod]` attributes
- Parameter extraction from step text
- Boolean assertions for Then steps
- Scenario Outlines with Examples tables

**Best for**: Learning the basics of file-based testing with TinyBDD
